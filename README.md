# Exercícios de Estrutura de Dados em C

Este repositório contém os exercícios desenvolvidos na disciplina de Estrutura de Dados.

# Organização

As atividades estão divididas em 3 pastas:

* **Pasta 1**
* **Pasta 2**
* **Pasta 3**

Dentro de cada pasta, os exercícios estão nomeados conforme a atividade proposta pelo professor no AVA.

# Exemplo:

* Exercicio1.c
* Exercicio2.c
* Exercicio3.c

# Observação

Na **Pasta 3**, os dois últimos exercícios estão interligados, ou seja, um depende do outro para funcionar corretamente.
